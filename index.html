Architecture posture

SSR-first, modular, provider-swappable

Astro pages provide the SSR shell, Fastify route modules own delivery, commands and queries stay explicit, and adapters isolate storage plus moderation providers.

Both web and API middleware enforce trusted-origin protections, CSP, auth boundaries, and locale propagation before critical writes land.

The storage layer exposes a hexagonal port through IStorageService, with the current production adapter resolved to Backblaze B2.
